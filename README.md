# Zentry - The Metagame Layer

A high-performance, visually immersive gaming landing page featuring modern Web3 aesthetics and advanced scroll-triggered animations.

## 🚀 Features

*   **Interactive Video Hero**: A dynamic hero section with a video switcher that cycles through gameplay trailers via the "Next" button.
*   **Scroll-Driven Animations**: 
    *   **autoBlur**: Elements smoothly blur and fade based on the user's scroll position.
    *   **autoTakeFull**: Images expand dynamically to fill the viewport as they enter the frame.
    *   **autoDisplay**: Content cards scale and slide into view using modern CSS scroll timelines.
*   **Animated Typography**: High-impact headers with animated linear-gradient text effects.
*   **Fully Responsive**: Optimized for all devices, from mobile phones to large desktops.
*   **Modern UI Components**: Glassmorphism navigation, custom clip-path image masks, and interactive hover states.

## 🛠️ Technologies Used

*   **HTML5**: Semantic structure for SEO and accessibility.
*   **CSS3**: Advanced animations using `animation-timeline: view()`, CSS Grid, and Flexbox.
*   **JavaScript (ES6+)**: Custom logic for the video carousel and interactive elements.
*   **Font Awesome**: Integrated iconography for social links.
